### Quickstart [![Gem Version](https://badge.fury.io/rb/dotenvx.svg)](https://badge.fury.io/rb/dotenvx)

Install and use it in code just like `dotenv`.

```sh
gem install dotenvx
```
```ruby
# index.rb
require "dotenvx/load"

puts "Hello #{ENV["HELLO"]}"
```
